# Qwen 2.5 설치 및 사용 가이드

한국어 → 영어 번역을 위한 Qwen 2.5 LLM 모델 설치 가이드

## 1. Ollama 설치

### macOS
```bash
# Homebrew로 설치 (추천)
brew install ollama

# 또는 공식 사이트에서 다운로드
# https://ollama.com/download
```

### Linux
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

### Windows
- https://ollama.com/download 에서 설치 프로그램 다운로드

## 2. Ollama 실행

```bash
# macOS/Linux: 백그라운드에서 자동 실행됨
# 수동 실행이 필요한 경우:
ollama serve
```

## 3. Qwen 2.5 모델 다운로드

```bash
# 7B 모델 (추천 - 약 4.7GB)
ollama pull qwen2.5:7b

# 더 작은 모델이 필요한 경우
ollama pull qwen2.5:3b

# 더 큰 모델 (성능 최고 - 약 8GB)
ollama pull qwen2.5:14b
```

## 4. 설치 확인

```bash
# 설치된 모델 확인
ollama list

# 간단한 테스트
ollama run qwen2.5:7b "안녕하세요를 영어로 번역해주세요"
```

## 5. Python으로 사용하기

### 5.1 Python 패키지 설치
```bash
pip install ollama
```

### 5.2 기본 사용 예제
```python
import ollama

# 간단한 번역
response = ollama.chat(model='qwen2.5:7b', messages=[
    {
        'role': 'user',
        'content': '이 코드를 리팩토링해주세요'
    }
])
print(response['message']['content'])
```

### 5.3 한→영 번역 전용 함수
```python
import ollama

def translate_ko_to_en(korean_text):
    """한국어를 영어로 번역"""
    response = ollama.chat(
        model='qwen2.5:7b',
        messages=[
            {
                'role': 'system',
                'content': 'You are a professional translator. Translate Korean to English accurately, preserving the original meaning and intent, especially for AI prompts and technical content.'
            },
            {
                'role': 'user',
                'content': f'Translate the following Korean text to English:\n\n{korean_text}'
            }
        ]
    )
    return response['message']['content']

# 사용 예제
korean_prompt = "사용자 인증 기능을 구현해주세요. JWT 토큰을 사용하고 Redis에 세션을 저장해야 합니다."
english_prompt = translate_ko_to_en(korean_prompt)
print(english_prompt)
```

### 5.4 파일 번역 스크립트
```python
import ollama

def translate_file(input_file, output_file):
    """txt 파일을 읽어서 번역 후 저장"""
    # 한글 파일 읽기
    with open(input_file, 'r', encoding='utf-8') as f:
        korean_text = f.read()

    # 번역
    response = ollama.chat(
        model='qwen2.5:7b',
        messages=[
            {
                'role': 'system',
                'content': 'Translate Korean to English for AI prompts.'
            },
            {
                'role': 'user',
                'content': f'Translate to English:\n\n{korean_text}'
            }
        ]
    )

    english_text = response['message']['content']

    # 영문 파일 저장
    with open(output_file, 'w', encoding='utf-8') as f:
        f.write(english_text)

    print(f"번역 완료: {input_file} → {output_file}")

# 사용
translate_file('prompt_kr.txt', 'prompt_en.txt')
```

## 6. 커맨드라인에서 직접 사용

```bash
# 대화형 모드
```

## 8. 시스템 요구사항

- **RAM**: 최소 8GB (7B 모델 기준)
- **저장공간**: 약 5GB
- **CPU**: x86_64 또는 ARM64
- **GPU**: 선택사항 (NVIDIA, AMD, Apple Silicon 가속 지원)

## 9. 문제 해결

### 모델 다운로드가 느린 경우
- 안정적인 네트워크 연결 확인
- 다시 시도: `ollama pull qwen2.5:7b`

### "connection refused" 오류
```bash
# Ollama 서버 재시작
ollama serve
```

### Python에서 연결 안 됨
```python
# 서버 주소 확인
import ollama
ollama.Client(host='http://localhost:11434')
```

## 10. 빠른 사용 - CLI 도구 (translate.py)

**가장 쉬운 방법! 명령어 한 줄로 파일 번역**

### 사용법
```bash
# 기본 사용
python translate.py input_kr.txt output_en.txt

# 예제
python translate.py prompt_kr.txt prompt_en.txt
```

### translate.py가 자동으로 해주는 일
1. 한글 파일 읽기
2. Qwen 2.5로 번역
3. 영문 파일 저장
4. 진행 상황 표시

### 설치
```bash
# ollama Python 패키지만 있으면 됨
pip install ollama
```

### 예제 실행
```bash
# 1. 한글 파일 만들기
echo "사용자 인증 기능을 구현해주세요" > test_kr.txt

# 2. 번역
python translate.py test_kr.txt test_en.txt

# 3. 결과 확인
cat test_en.txt
```
